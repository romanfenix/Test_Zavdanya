# Varus AI Product Matcher 🛒

Це рішення тестового завдання для позиції Python-розробника. 
Програма автоматично аналізує список покупок у вільному форматі, знаходить найрелевантніші товари на сайті супермаркету **VARUS** в обхід анти-бот систем та розраховує оптимальну кількість упаковок для мінімального покриття потреби користувача.

## 🌟 Основні можливості (Features)

1. **AI Парсинг (OpenAI + Pydantic):** Перетворення неструктурованого тексту (наприклад, "середній банан, 2 шт.") у структуровані дані (назва, кількість, одиниці виміру, характеристики).
2. **Інтеграція пошуку (Google Custom Search API):** Стабільний пошук товарів по каталогу `varus.ua` без ризику блокувань від Cloudflare (403/429 помилок).
3. **Парсинг сторінок товарів:** Використання `httpx` та `BeautifulSoup` для витягування ваги, ціни та формату продажу (поштучно/на вагу) з HTML та JSON-LD розмітки сайту.
4. **AI Ранжування:** Використання LLM для аналізу знайдених товарів та вибору найбільш релевантного (наприклад, відсіювання бананових нектарів чи масок для волосся при пошуку свіжих бананів).
5. **Алгоритм розрахунку:** Математичне обчислення необхідної кількості упаковок для задоволення потреби з мінімальним надлишком.

## 🛠 Технологічний стек

- **Python 3.10+**
- `httpx` — для асинхронних / синхронних HTTP-запитів.
- `beautifulsoup4` — для парсингу HTML.
- `openai` — для взаємодії з LLM.
- `pydantic` — для суворої типізації та Structured Outputs.
- `python-dotenv` — для безпечної роботи з API-ключами.

## 🚀 Інструкція із запуску

### 1. Клонування та налаштування середовища
Склонуйте репозиторій та перейдіть у папку проєкту:
```bash
git clone <посилання-на-ваш-репозиторій>
cd <назва-папки>

### 2. Створіть та активуйте віртуальне середовище:
python -m venv venv

# Для Windows:
venv\Scripts\activate
# Для Linux/macOS:
source venv/bin/activate

### 3. Встановіть залежності:
pip install -r requirements.txt

### 4. Створіть файл .env у корені проєкту та додайте туди наступні ключі:
GITHUB_TOKEN=ваш_токен_github_models_або_openai
GOOGLE_API_KEY=ваш_ключ_google_custom_search
GOOGLE_CX=ваш_ідентифікатор_пошукової_системи

🔑 Як отримати API ключі:

GITHUB_TOKEN / OpenAI API Key: Можна використовувати стандартний ключ OpenAI (sk-...) або безкоштовний GitHub Models Token (Personal Access Token).

GOOGLE_API_KEY: Перейдіть у Google Cloud Console, увімкніть Custom Search API у розділі APIs & Services -> Library. Потім створіть API Key у розділі Credentials.

GOOGLE_CX: Перейдіть у Programmable Search Engine, створіть пошукову систему, вказавши сайт varus.ua, та скопіюйте її "Ідентифікатор пошукової системи".

### 5. Використання:
Додайте ваші продукти у файл products.txt (кожен продукт з нового рядка).
Формат: <назва продукту>, <кількість>

Запустіть програму:
python main.py
